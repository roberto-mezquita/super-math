# Super Math

Aplicación educativa desarrollada mediante herramientas No-Code con el objetivo de fortalecer el aprendizaje de las matemáticas a través de ejercicios interactivos y espacios colaborativos.

## Descripción

Super Math fue diseñado para apoyar a estudiantes en la práctica de operaciones matemáticas y fomentar el aprendizaje colaborativo mediante comunidades y foros integrados dentro de la plataforma.

## Características

* Ejercicios matemáticos interactivos
* Comunidades de aprendizaje
* Foros de participación estudiantil
* Diseño orientado a la experiencia del usuario
* Desarrollo utilizando herramientas No-Code
